# 目標平台
| Platform | DSD mode | Output Device Type |
|----------|----------|----------|
| Windows | Native DSD | ASIO |
| Windows | DSD to PCM | WSAPI Shared Mode |
| Windows | DSD to PCM | WSAPI Exculisive Mode |
| MAC OSX | DOP | CoreAudio |
