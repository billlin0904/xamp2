# XAMP 2

# GOAL
- 此程式純屬練功娛樂用 :)
- C++17
- 專注本地端播放體驗

# 目標平台
| Platform | DSD mode | Output Device Type |
|----------|----------|----------|
| Windows | Native DSD | ASIO |
| Windows | DSD to PCM | WSAPI (Exculisive Mode / Shared Mode) |
| MAC OSX | DOP | CoreAudio |
| Windows | DOP | WSAPI Exculisive Mode |
